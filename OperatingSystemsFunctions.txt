One function of an operating system is having a UI. a UI (user interface) controls how data is shown and entered. a GUI (graphical user interface) uses visual images to display menus and other parts of the UI.
Another function of an operating system is handling memory. When handling memory, the OS is also in charge of optimizing internal memory. The OS can use physical or virtual memory. 
Most OSes have a function to connect to the internet. This function allows the computer to make use of a NIC (network interface card) to connect wirelessly, an ethernet cable, or usb device to establish a connection.
